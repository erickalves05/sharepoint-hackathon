## Description

**My Work Hub** is a SharePoint Framework web part that brings tasks, meetings, approvals, recent files, and an AI summary into a single hub.

**Features:**

- **Tasks tab**: Unified view of Microsoft To Do and Planner tasks, including flagged emails. Source icons (Outlook, To Do, Planner), due date badges (Today/Tomorrow), plan/list names, and one-click completion.
- **Meetings tab**: Next 5 upcoming calendar events with date badges (Today/Tomorrow), organizer, time range, and link to open the meeting.
- **Approvals tab**: Pending approvals with markdown descriptions, requester photo and name, and a reject-with-reason dialog.
- **Recent tab**: Recent OneDrive files with file-type icons (Excel, Word, PowerPoint, PDF, etc.), creator, and last modified date.
- **Summary tab**: Microsoft 365 Copilot summary of pending work, using tasks, approvals, and recent files as context, plus actionable recommendations.

**Capabilities used:** Microsoft Graph (To Do, Planner, Drive, Users, Calendar), Graph Beta (Approvals, Copilot Chat), SPFx 1.21.1, Fluent UI, React.
